# Analog Gauge Reader

Reads analog pressure gauges using computer vision (OpenCV).

## Features
- Works with any Home Assistant camera entity
- Configurable update intervals (1 min / 15 min)
- Multi-level alarm thresholds

## Installation via HACS
1. Add this repository as a custom repository in HACS
2. Search for "Analog Gauge Reader"
3. Install and restart Home Assistant
